# 🚗 Car Price Prediction using Machine Learning

## 📌 Overview
This project predicts the price of used cars based on features like year, mileage, fuel type, and engine size.

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn

## 🤖 Models Used
- Linear Regression
- Lasso, Ridge
- Decision Tree
- Random Forest
- Support Vector Machine

## ⚙️ Features
- Data preprocessing
- Model training
- Model evaluation (MAE, RMSE, R²)
- Hyperparameter tuning
- Model saving

## ▶️ How to Run
pip install -r requirements.txt  
python src/model.py

## 📊 Result
Random Forest gave the best performance.
